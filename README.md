# Lista-1-em-Java

1. Escreva um algoritmo que exiba 20 vezes a mensagem “Eu gosto de
programar!!!”

a) Utilize o comando de repetição Enquanto (While).

b)Utilize o comando de repetição Repita (Do...While).

c)Utilize o comando de repetição Para (For).


2. Construa um algoritmo que leia do usuário o salário e exiba uma mensagem
de acordo com as seguintes condições: se o salário for até R$ 645, escreva a
mensagem “Até 1 salário-mínimo; se o salário for acima de R$ 645 e até R$
1935, escreva a mensagem “Até 3 salários mínimos”; se for acima de R$ 1935
e abaixo deR$ 3225, escreva a mensagem “Até 5 salários mínimos”; se for
acima de R$ 3225, escreva a mensagem “Acima de 5 salários mínimos”.

3. Escreva um algoritmo que receba um determinado número inteiro pelo
usuário e imprima informando se é um numero par ou impar e se é um número
positivo ou negativo. Zero é o único número que não pode ser verificado.


4. Escreva um algoritmo que leia uma sequência de números do usuário e
realize a soma desses números. Encerre a execução quando um número
negativo for digitado.


5. Implemente um jogo de adivinhação em que o computador escolhe um
número aleatório entre 1 e 100, e o jogador deve adivinhar. O programa deve:

✓ Permitir que o usuário insira palpites

✓ Informar se o palpite está alto ou baixo em relação ao número correto

✓ Contar o número de tentativas

✓ Exibir uma mensagem de parabéns quando o usuário acertar

✓ Dica: Use a classe Randon

o Random random = new Random();

o int numeroSecreto = random.nextInt(100) + 1//considera de 1 até
100

6. Crie um programa que simule uma votação entre três candidatos. O programa
NÃO deve USAR VETORES.

• Exibir os candidatos e permitir que o usuário vote digitando um número
correspondente

• Contabilizar os votos

• Permitir que vários usuários votem até que a opção de encerrar a votação
seja escolhida

• Exibir o total de votos para cada candidato e o vencedor


7. Escreva um algoritmo que leia o ano de nascimento de uma pessoa e
apresente uma linha do tempo, mostrando cada ano de vida da pessoa e
quantos anos ela tinha em cada ano até o ano atual.


8. Validação de Acesso Crie um sistema que peça o nome de usuário e a senha.
O usuário correto é "coordenador" e a senha é "java2026". A verificação do
nome não deve diferenciar maiúsculas de minúsculas (equalsIgnoreCase),
mas a senha deve ser exata (equals). Exiba mensagens específicas para: login
com sucesso, senha incorreta ou usuário não encontrado.


9. A classe Math em Java possui métodos prontos para cálculos matemáticos
e não precisa ser importada, pois já pertence ao pacote padrão da
linguagem.

Crie um programa que leia dois números reais e exiba: o maior valor
(Math.max), o menor valor (Math.min), a potência do primeiro pelo segundo
(Math.pow), a raiz quadrada do primeiro número (Math.sqrt) e o valor
absoluto da diferença entre eles (Math.abs).
